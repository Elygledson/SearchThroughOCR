# SearchThroughOCR

## To use this repository, you need to create an account on Google Api and openAI to get an access key in this way you can use correctly and take advantage of searching in pdf files or images to create spreadsheet from them
## Extract text from image or pdf 
## Case you are using extracting text from images type command below:
## python search.py -i [image-name|images-folder]
## Case you are using  pdf files type command below:
## python search.py -f [file name|folder name]

## Observation: you should use the forward-slash when you are typing folder name
## Example: c:/folderName
